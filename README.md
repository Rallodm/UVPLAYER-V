# UVPlayer Landing Page (Next.js + Tailwind)

This is a minimal starter repo for the UVPlayer landing page.
- Next.js + Tailwind CSS
- Dark-themed marketing landing page for UVPlayer

## Quick start (locally)

1. Install deps:
```bash
npm install
```

2. Run dev server:
```bash
npm run dev
```

Notes:
- Tailwind must be configured. This repo includes `tailwind.config.js` and `postcss.config.js`.
- Replace the mockup images and logo SVG with your real assets in `pages/index.js`.
